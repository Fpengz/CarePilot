# Design Principles

This document captures core product design principles for CarePilot. It complements `docs/design-docs/index.md`.

## Core Beliefs
- Design should improve clinician insight quality and patient engagement.
- UX must preserve safety-critical clarity over novelty.
- Each feature should map to measurable health impact.
