# Blood Pressure Graphs Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add blood pressure trend charts to the Companion BP card and the Dashboard so BP data is visible at a glance.

**Architecture:** Introduce a lightweight BP time-series endpoint (per user) that reads paired BP metrics and returns chart-ready buckets. Reuse a shared chart component to render a small sparkline-style graph in the Companion BP card and a full chart in the Dashboard. Keep all BP computation deterministic and in backend services.

**Tech Stack:** FastAPI, Pydantic, React (Next.js), Recharts, TypeScript.
