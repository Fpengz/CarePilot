# Planning Standards

Plans are first-class artifacts. Use execution plans for complex work, lightweight checklists for small changes.

## Rules
- Active plans live in `docs/exec-plans/active/`.
- In-progress work lives in `docs/exec-plans/in-progress/`.
- Completed plans live in `docs/exec-plans/completed/`.
- Always update `docs/exec-plans/index.md` when a plan moves state.
