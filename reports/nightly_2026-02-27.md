## Architecture Decisions
- Decision: preserve `/reports` route as redirect instead of deleting it
  - Why: keeps old links/bookmarks stable while migrating UX to suggestions-first flow
  - Tradeoff: one extra compatibility route to maintain
- Decision: suggestions page uses persisted history from `/api/v1/suggestions`
  - Why: aligns frontend with backend source of truth and supports future household scope extension
  - Tradeoff: still keeps JSON viewer as secondary debug output for now
